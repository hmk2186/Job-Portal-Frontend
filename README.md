# 🚀 Job Portal Frontend

A modern, responsive, and feature-rich Job Portal Frontend built using **HTML, CSS, JavaScript, and Bootstrap 5**. The platform provides an intuitive experience for both job seekers and employers with dynamic job listings, company profiles, dashboards, dark mode support, and responsive design.

## 🌐 Live Demo

🔗 **Live Website:**
https://job-portal-frontend-puce-ten.vercel.app/

---

# 📌 Features

## 👤 Job Seeker Features

* Browse available job opportunities
* Search jobs by keyword
* Filter jobs by location, category, experience, and salary
* View detailed job information
* Save and bookmark jobs
* Explore companies and their profiles
* Responsive user interface
* Dark Mode support

---

## 🏢 Employer Features

* Employer Dashboard
* Manage posted jobs
* View applicants
* Dashboard statistics and analytics
* Job management interface

## 📊 Candidate Dashboard
